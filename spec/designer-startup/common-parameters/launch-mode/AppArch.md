# AppArch

- Вид: `parameter`
- Группа: Определение режима запуска
- Source pagePath: `zif2_AppArch`
- Source URL: `http://localhost:8080/ru/1%D0%A1%3A%D0%9F%D1%80%D0%B5%D0%B4%D0%BF%D1%80%D0%B8%D1%8F%D1%82%D0%B8%D0%B5?page=zif2_AppArch`

## Синтаксис

```text
/AppArch <разрядность>
```

## Нормализованное описание

— указывает разрядность используемого клиентского приложения на 64-разрядных ОС семейства Windows.

Параметр **<разрядность>** может принимать следующие значения:

- x86 — использовать только 32-разрядные версии,

- x86_64 — использовать только 64-разрядные версии,

- х86_prt — поиск самой актуальной версии, и если для найденной версии доступны приложения в обоих вариантах разрядности, выбирается 32-разрядная версия,

- x86_64_prt — поиск самой актуальной версии, и если для найденной версии доступны приложения в обоих вариантах разрядности, выбирается 64-разрядная версия.
