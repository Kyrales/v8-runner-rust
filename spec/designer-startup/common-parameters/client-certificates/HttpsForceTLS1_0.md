# HttpsForceTLS1_0

- Вид: `parameter`
- Группа: Использование клиентских сертификатов (только для тонкого клиента)
- Source pagePath: `zif2_HttpsForceTLS1_0`
- Source URL: `http://localhost:8080/ru/1%D0%A1%3A%D0%9F%D1%80%D0%B5%D0%B4%D0%BF%D1%80%D0%B8%D1%8F%D1%82%D0%B8%D0%B5?page=zif2_HttpsForceTLS1_0`

## Синтаксис

```text
/HttpsForceTLS1_0
```

## Нормализованное описание

— определяет принудительное использование протокола TLS версии 1.0 системой
"1С:Предприятие" при работе через веб-сервер с помощью протокола
HTTPS.
**Внимание!** Запрещено одновременное использование
нескольких параметров /HttpsForce* (например, /HttpsForceTLS1_0 и
/HttpsForceTLS1_2). При одновременном указании поведение не определено. Если в
командной строке отсутствуют команды /HttpsForceTLS*, то клиентское приложение
пытается использовать протокол TLS с версии 1.3 (понижая версию протокола, если
того требует используемый веб-сервер).
