# EnableCheckModal

- Вид: `parameter`
- Группа: Проверки во время работы клиентского приложения
- Source pagePath: `zif2_EnableCheckModal`
- Source URL: `http://localhost:8080/ru/1%D0%A1%3A%D0%9F%D1%80%D0%B5%D0%B4%D0%BF%D1%80%D0%B8%D1%8F%D1%82%D0%B8%D0%B5?page=zif2_EnableCheckModal`

## Синтаксис

```text
/EnableCheckModal
```

## Нормализованное описание

— предназначен для запуска в режиме "строгой" проверки использования модальных методов.  В конфигурации, для которой режим модальности установлен в значение "Не использовать" или "Использовать с предупреждениями", при вызове модального метода будет сгенерирована исключительная ситуация или отображено предупреждение.
