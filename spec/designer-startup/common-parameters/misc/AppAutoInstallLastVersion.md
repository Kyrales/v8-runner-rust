# AppAutoInstallLastVersion

- Вид: `parameter`
- Группа: Прочие параметры
- Source pagePath: `zif2_appautoinstalllastversion`
- Source URL: `http://localhost:8080/ru/1%D0%A1%3A%D0%9F%D1%80%D0%B5%D0%B4%D0%BF%D1%80%D0%B8%D1%8F%D1%82%D0%B8%D0%B5?page=zif2_appautoinstalllastversion`

## Синтаксис

```text
/AppAutoInstallLastVersion+/-
```

## Нормализованное описание

— управляет автоматической установкой новых версий приложения:

"+" — установка новых версий включена;

"-" — установка новых версий выключена.
