# DisableUnrecoverableErrorMessage

- Вид: `parameter`
- Группа: Прочие параметры
- Source pagePath: `DisableUnrecoverableErrorMessage`
- Source URL: `http://localhost:8080/ru/1%D0%A1%3A%D0%9F%D1%80%D0%B5%D0%B4%D0%BF%D1%80%D0%B8%D1%8F%D1%82%D0%B8%D0%B5?page=DisableUnrecoverableErrorMessage`

## Синтаксис

```text
/DisableUnrecoverableErrorMessage
```

## Нормализованное описание

— если данный параметр указан, то при возникновении невосстановимой ошибки (в том числе, аварийного завершения):

- Окно ошибки показано не будет.

- При аварийном завершении дамп сформируется в соответствии с настройками в logcfg.xml

- Отчет об ошибке будет отправлен, если настроена автоматическая отправка и размер отчета не превышает ограничение на отправку без вопроса о большом размере отчета (до 5 Мбайт).

- Приложение будет завершено с кодом возврата, отличным от нуля.

Рекомендуется указывать этот параметр, если требуется запустить клиентское приложение в окружении, где не будет пользователя, который отреагирует на сообщение о невосстановимой ошибке.
