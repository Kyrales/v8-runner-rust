# DisableSplash

- Вид: `parameter`
- Группа: Прочие параметры
- Source pagePath: `zif2_DisableSplash`
- Source URL: `http://localhost:8080/ru/1%D0%A1%3A%D0%9F%D1%80%D0%B5%D0%B4%D0%BF%D1%80%D0%B8%D1%8F%D1%82%D0%B8%D0%B5?page=zif2_DisableSplash`

## Синтаксис

```text
/DisableSplash
```

## Нормализованное описание

— отключает отображение заставки при запуске "1С:Предприятия", если она была полностью заменена.

Для того, чтобы полностью заменить заставку, необходимо обращаться в фирму "1С".
