# TestClient

- Вид: `parameter`
- Группа: Настройки тестирования
- Source pagePath: `zif2_testclient`
- Source URL: `http://localhost:8080/ru/1%D0%A1%3A%D0%9F%D1%80%D0%B5%D0%B4%D0%BF%D1%80%D0%B8%D1%8F%D1%82%D0%B8%D0%B5?page=zif2_testclient`

## Синтаксис

```text
/TestClient [-TPort<Номер TCP-порта>]
```

## Нормализованное описание

— запуск толстого и тонкого клиента как управляемого другими клиентами с помощью менеджера тестирования.

**-TPort<Номер TCP-порта>** — указывает номер порта для взаимодействия клиента и менеджера тестирования. По умолчанию используется порт 1538.
