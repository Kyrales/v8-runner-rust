# VL

- Вид: `parameter`
- Группа: Настройки локализации
- Source pagePath: `zif2_vl`
- Source URL: `http://localhost:8080/ru/1%D0%A1%3A%D0%9F%D1%80%D0%B5%D0%B4%D0%BF%D1%80%D0%B8%D1%8F%D1%82%D0%B8%D0%B5?page=zif2_vl`

## Синтаксис

```text
/VL <код локализации сеанса>
```

## Нормализованное описание

— указывается код локализации сеанса, используемый при форматировании данных типа Число и Дата, а также в методах ЧислоПрописью() и ПредставлениеПериода().
