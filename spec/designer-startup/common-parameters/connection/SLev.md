# SLev

- Вид: `parameter`
- Группа: Указание параметров подключения
- Source pagePath: `zif2_slev`
- Source URL: `http://localhost:8080/ru/1%D0%A1%3A%D0%9F%D1%80%D0%B5%D0%B4%D0%BF%D1%80%D0%B8%D1%8F%D1%82%D0%B8%D0%B5?page=zif2_slev`

## Синтаксис

```text
/SLev<Уровень>
```

## Нормализованное описание

— определяет уровень защищенности соединения клиента с сервером "1С:Предприятия" (используется для тонкого и веб-клиентов в режиме "1С:Предприятие").

Параметр <Уровень> может принимать следующие значения:
0 — незащищенное соединение;
1 — защищенное соединение только в процессе выполнения аутентификации;
2 — защищенное соединение в течение всего сеанса;
Если не указан, используется значение 0.
