# O

- Вид: `parameter`
- Группа: Указание параметров подключения
- Source pagePath: `zif2_o`
- Source URL: `http://localhost:8080/ru/1%D0%A1%3A%D0%9F%D1%80%D0%B5%D0%B4%D0%BF%D1%80%D0%B8%D1%8F%D1%82%D0%B8%D0%B5?page=zif2_o`

## Синтаксис

```text
/O <скорость соединения>
```

## Нормализованное описание

— определяет скорость соединения (используется в **тонком** клиенте):

**Normal** — обычная,

**Low** — низкая скорость соединения.
