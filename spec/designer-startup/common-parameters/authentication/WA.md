# WA

- Вид: `parameter`
- Группа: Настройка аутентификации
- Source pagePath: `zif2_wa`
- Source URL: `http://localhost:8080/ru/1%D0%A1%3A%D0%9F%D1%80%D0%B5%D0%B4%D0%BF%D1%80%D0%B8%D1%8F%D1%82%D0%B8%D0%B5?page=zif2_wa`

## Синтаксис

```text
/WA [+/-]
```

## Нормализованное описание

— определяет режим использования аутентификации операционной системы при запуске "1С:Предприятия". Если параметр /WA не указывается, то подразумевается, что используется параметр командной строки /WA+.

**/WA-** — запрет применения аутентификации операционной системы при старте "1С:Предприятия".

**/WA+** — установка обязательного применения аутентификации операционной системы при старте "1С:Предприятия".
