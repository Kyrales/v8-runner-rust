# WSP

- Вид: `parameter`
- Группа: Настройка аутентификации
- Source pagePath: `zif2_wsp`
- Source URL: `http://localhost:8080/ru/1%D0%A1%3A%D0%9F%D1%80%D0%B5%D0%B4%D0%BF%D1%80%D0%B8%D1%8F%D1%82%D0%B8%D0%B5?page=zif2_wsp`

## Синтаксис

```text
/WSP <пароль>
```

## Нормализованное описание

— пароль пользователя, имя которого указано в параметре /WSN, для аутентификации на веб-сервере.
