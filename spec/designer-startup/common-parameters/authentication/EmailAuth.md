# EmailAuth

- Вид: `parameter`
- Группа: Настройка аутентификации
- Source pagePath: `zif2_EmailAuth`
- Source URL: `http://localhost:8080/ru/1%D0%A1%3A%D0%9F%D1%80%D0%B5%D0%B4%D0%BF%D1%80%D0%B8%D1%8F%D1%82%D0%B8%D0%B5?page=zif2_EmailAuth`

## Синтаксис

```text
/EmailAuth <email>
```

## Нормализованное описание

— позволяет выполнить аутентификацию через электронную почту. На указанную электронную почту будет отправлен код подтверждения и будет открыто окно для ввода этого кода.

Данная команда не поддерживается для режима пакетного запуска конфигуратора, а также в том случае, если конфигуратор запускается в режиме, запрещающем отображение диалогов (/DisableStartupDialogs).
