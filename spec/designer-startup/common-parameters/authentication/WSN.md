# WSN

- Вид: `parameter`
- Группа: Настройка аутентификации
- Source pagePath: `zif2_wsn`
- Source URL: `http://localhost:8080/ru/1%D0%A1%3A%D0%9F%D1%80%D0%B5%D0%B4%D0%BF%D1%80%D0%B8%D1%8F%D1%82%D0%B8%D0%B5?page=zif2_wsn`

## Синтаксис

```text
/WSN <имя>
```

## Нормализованное описание

– Имя пользователя операционной системы, которое будет использовано системой для выполнения аутентификации на веб-сервере в случае указания параметра /WSA+. Пароль пользователя указывается в параметре /WSP.
