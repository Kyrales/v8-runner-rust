# DumpResult

- Вид: `parameter`
- Раздел: Прочие параметры
- Путь в оглавлении: Прочие параметры -> DumpResult
- Source pagePath: `zif3_dumpresult`
- Source URL: `http://localhost:8080/ru/1%D0%A1%3A%D0%9F%D1%80%D0%B5%D0%B4%D0%BF%D1%80%D0%B8%D1%8F%D1%82%D0%B8%D0%B5?page=zif3_dumpresult`

## Синтаксис

```text
/DumpResult <имя файла>
```

## Нормализованное описание

— предназначен для записи результата работы конфигуратора в файл. После параметра должно быть указано имя файла. Результат - число (0 - в случае успеха).

**Внимание!** Если в параметре командной строки требуется ввести имя файла, то следует учитывать, что при указании имени файла с полным путем все каталоги, входящие в состав пути, должны существовать.
