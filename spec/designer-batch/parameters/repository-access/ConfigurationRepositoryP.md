# ConfigurationRepositoryP

- Вид: `parameter`
- Раздел: Команды работы с хранилищем конфигурации
- Путь в оглавлении: Команды работы с хранилищем конфигурации -> Параметры доступа к хранилищу -> ConfigurationRepositoryP
- Source pagePath: `zif3_configurationrepositoryp`
- Source URL: `http://localhost:8080/ru/1%D0%A1%3A%D0%9F%D1%80%D0%B5%D0%B4%D0%BF%D1%80%D0%B8%D1%8F%D1%82%D0%B8%D0%B5?page=zif3_configurationrepositoryp`

## Синтаксис

```text
/ConfigurationRepositoryP <пароль>
```

## Нормализованное описание

— указание пароля пользователя хранилища. Если при создании хранилища для /ConfigurationRepositoryCreate** **был указан параметр **CheckPasswordComplexity**, то будет выполнена проверка пароля на соответствие политике.
