# RollbackCfg

- Вид: `command`
- Раздел: Конфигурация и расширения
- Путь в оглавлении: Конфигурация и расширения -> RollbackCfg
- Source pagePath: `zif3_rollbackcfg`
- Source URL: `http://localhost:8080/ru/1%D0%A1%3A%D0%9F%D1%80%D0%B5%D0%B4%D0%BF%D1%80%D0%B8%D1%8F%D1%82%D0%B8%D0%B5?page=zif3_rollbackcfg`

## Синтаксис

```text
/RollbackCfg [-Extension <имя расширения>]
```

## Нормализованное описание

— возврат к конфигурации базы данных. Доступные параметры:

**Extension <Имя расширения>** — обработка расширения с указанным именем. Если расширение успешно обработано возвращает код возврата 0, в противном случае (если расширение с указанным именем не существует или в процессе работы произошли ошибки) — 1.
