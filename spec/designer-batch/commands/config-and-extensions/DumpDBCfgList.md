# DumpDBCfgList

- Вид: `command`
- Раздел: Конфигурация и расширения
- Путь в оглавлении: Конфигурация и расширения -> DumpDBCfgList
- Source pagePath: `zif3_dumpdbcfglist`
- Source URL: `http://localhost:8080/ru/1%D0%A1%3A%D0%9F%D1%80%D0%B5%D0%B4%D0%BF%D1%80%D0%B8%D1%8F%D1%82%D0%B8%D0%B5?page=zif3_dumpdbcfglist`

## Синтаксис

```text
/DumpDBCfgList [-Extension <имя расширения>] [-AllExtensions]
```

## Нормализованное описание

— вывод расширений конфигурации. Если ни один параметр не задан, будет выдана ошибка. Доступные параметры:

**Extension <Имя расширения>** — выводит имя конфигурации-расширения. Если расширение не существует, будет выдана ошибка;
**AllExtensions** — выводит имена всех расширений.
