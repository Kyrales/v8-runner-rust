## Синтаксис

```text
/MobileAppWriteFile <имя файла>
```

## Нормализованное описание

— cохранить конфигурацию в ZIP-файл. Указанный файл используется для сборки приложения для мобильного устройства. ZIP-файл содержит описание конфигурации (в виде XML-файла) и файл с данными, которые используются для создания информационной базы на мобильном устройстве.

**Внимание!** Если в параметре командной строки требуется ввести имя файла, то следует учитывать, что при указании имени файла с полным путем все каталоги, входящие в состав пути, должны существовать.
