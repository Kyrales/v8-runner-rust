# ConfigurationRepositoryDumpCfg

- Вид: `command`
- Раздел: Команды работы с хранилищем конфигурации
- Путь в оглавлении: Команды работы с хранилищем конфигурации -> Операции со всей конфигурацией -> ConfigurationRepositoryDumpCfg
- Source pagePath: `zif3_configurationrepositorydumpcfg`
- Source URL: `http://localhost:8080/ru/1%D0%A1%3A%D0%9F%D1%80%D0%B5%D0%B4%D0%BF%D1%80%D0%B8%D1%8F%D1%82%D0%B8%D0%B5?page=zif3_configurationrepositorydumpcfg`

## Синтаксис

```text
/ConfigurationRepositoryDumpCfg [-Extension <имя расширения>] <имя cf файла> [-v <номер версии хранилища>]
```

## Нормализованное описание

— сохранить конфигурацию из хранилища в файл (пакетный режим запуска). Доступны параметры:

**-Extension <имя расширения>** — Имя расширения. Если параметр не указан, выполняется попытка соединения с хранилищем основной конфигурации, и команда выполняется для основной конфигурации. Если параметр указан, выполняется попытка соединения с хранилищем указанного расширения, и команда выполняется для этого хранилища.

**-v <номер версии хранилища>** v — Номер версии, если номер версии не указан, или равен -1, будет сохранена последняя версия.
