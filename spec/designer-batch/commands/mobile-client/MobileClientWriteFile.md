# MobileClientWriteFile

- Вид: `command`
- Раздел: Мобильный клиент
- Путь в оглавлении: Мобильный клиент -> MobileClientWriteFile
- Source pagePath: `zif3_MobileClientWriteFile`
- Source URL: `http://localhost:8080/ru/1%D0%A1%3A%D0%9F%D1%80%D0%B5%D0%B4%D0%BF%D1%80%D0%B8%D1%8F%D1%82%D0%B8%D0%B5?page=zif3_MobileClientWriteFile`

## Синтаксис

```text
/MobileClientWriteFile <имя файла>
```

## Нормализованное описание

— записывает xml-файл мобильного клиента с заданным именем. Возможно предварительное обновление конфигурации базы данных.

**Внимание!** Если в параметре командной строки требуется ввести имя файла, то следует учитывать, что при указании имени файла с полным путем все каталоги, входящие в состав пути, должны существовать.
