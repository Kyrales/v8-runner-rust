## Синтаксис

```text
/ManageCfgSupport [-disableSupport] [-force]
```

## Нормализованное описание

— предназначен для управления настройками поддержки конфигурации. Допустимо использование следующих параметров:

**disableSupport **— признак необходимости снятия конфигурации с поддержки. Если не указан, в файл протокола будет выведено сообщение об ошибке.

**force **— используется для снятия конфигурации с поддержки даже в том случае, если в конфигурации не разрешены изменения. Если не указан, а в конфигурации на момент выполнения команды не разрешены изменения, конфигурация не будет снята с поддержки, а в файл протокола будет выведено сообщение об ошибке.
