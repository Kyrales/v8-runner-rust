# DumpIB

- Вид: `command`
- Раздел: Выгрузка и загрузка информационной базы
- Путь в оглавлении: Выгрузка и загрузка информационной базы -> DumpIB
- Source pagePath: `zif3_dumpib`
- Source URL: `http://localhost:8080/ru/1%D0%A1%3A%D0%9F%D1%80%D0%B5%D0%B4%D0%BF%D1%80%D0%B8%D1%8F%D1%82%D0%B8%D0%B5?page=zif3_dumpib`

## Синтаксис

```text
/DumpIB <имя файла>
```

## Нормализованное описание

— выгрузка информационной базы в командном режиме.

**Внимание!** Если в параметре командной строки требуется ввести имя файла, то следует учитывать, что при указании имени файла с полным путем все каталоги, входящие в состав пути, должны существовать.
